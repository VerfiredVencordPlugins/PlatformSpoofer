# Platform Spoofer

Discord'da hangi cihazdan bağlı göründüğünü değiştiren Vencord plugin'i.

## Özellikler

| Platform | Görünüm |
|----------|---------|
| Desktop | Discord Client |
| Web | Chrome |
| Mobile | Discord iOS |
| Embedded | Discord Embedded (Konsol) |

## Kurulum

### UserpluginInstaller ile (Önerilen)
```
https://github.com/VerfiredVencordPlugins/PlatformSpoofer
```
Linki Discord'a yapıştır ve "Install Plugin" butonuna tıkla.

### Manuel
```bash
cd Vencord/src/userplugins
git clone https://github.com/VerfiredVencordPlugins/PlatformSpoofer.git
cd ../..
pnpm build
```

## Kullanım

1. Vencord ayarlarına git
2. Platform Spoofer'ı aktif et
3. Platform seç (Desktop, Web, Mobile, Embedded)
4. Discord'u yeniden başlat
